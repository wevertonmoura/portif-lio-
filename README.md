# 🚀 Portfolio Elite v1.0 | Weverton Moura

Este é o meu portfólio profissional, desenvolvido com foco em **UI/UX Design de Alta Performance** e **Desenvolvimento Full-Stack**. O projeto utiliza técnicas avançadas de interface moderna para proporcionar uma experiência imersiva e responsiva.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as melhores práticas de desenvolvimento front-end:

* **HTML5 & CSS3 (Moderno):** Uso de variáveis CSS, Flexbox e Grid Layout.
* **JavaScript (ES6+):** Implementação de micro-interações, efeitos de digitação e lógica de scroll.
* **AOS (Animate On Scroll):** Para animações suaves de surgimento de conteúdo.
* **Phosphor Icons:** Biblioteca de ícones minimalistas e modernos.
* **Glassmorphism & Neon Effects:** Estética baseada em transparências (vidro fosco) e luzes dinâmicas.

---

## ✨ Funcionalidades "Elite" Implementadas

* 🎯 **Efeito Spotlight:** Cards que reagem ao movimento do mouse com iluminação dinâmica.
* ⌨️ **Typing Effect:** Texto de apresentação que se auto-digita, alternando entre as especialidades.
* 🖱️ **Magnetic Buttons:** Botões com interação física sutil (atração magnética ao cursor).
* 📱 **Menu Mobile Premium:** Navegação totalmente responsiva com menu hambúrguer animado.
* 📋 **Fast-Action E-mail:** Função de cópia automática do e-mail para a área de transferência com um clique.
* 🌧️ **Falling Icons Background:** Fundo animado com ícones tecnológicos caindo via CDN (Devicon).
